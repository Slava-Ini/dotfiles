## Commands

`ch` is an alias to `chezmoi` (set in `config.fish` and PowerShell startup script)

- `ch apply` - apply diffs to local configuration from chezmoi repo
   - `--exclude=scripts` exclude scripts
- `ch diff` - show difference between local configration and chzmoi local repo
- `ch ignore` - display what files are ignored on the current system according to `.chezmoiignore`
